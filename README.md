Android Commons Library
=======================

Shared Android Library

**Features**

Helpers for

* Analytics
* Datastore
* Function extensions for
  * Context
  * Enum
  * Flow
  * Map
  * SavedStateHandle
  * java.time.Date
* Firebase
  * Firestore
  * Remote Config
* Kotlin json SerializationFactory
* Logging
* Networking
